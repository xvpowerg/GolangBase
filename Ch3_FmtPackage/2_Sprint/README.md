# Sprintf
可使用特定格式輸出字串
## [常用格式如下](https://golang.org/pkg/fmt/)
| 格式        | 功能   |  
| --------   | -----:  |
| %T        |   輸出類型   | 
| %t        |    輸出bool   |
| %d        |    輸出整數   |
| %f        |    輸出浮點數   |
| %s        |    輸出字串   |
| %p        |   可輸出指標並用16進位表示   |
| %%        |   輸出%   |
| %v        | 自動判定類型輸出   |
