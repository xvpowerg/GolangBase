# 資料類型與宣告
## 資料類型
### [基本型態](https://golang.org/ref/spec#Numeric_types)
+ 整數
  + int8, int16, int32, int64
  + uint8, uint16, uint32, uint64
+ 浮點數
  + float32
  + float64
+ 複數
  + complex64
  + complex128
+ Boolane
```go
//可使用var 直接指定類型宣告
//或是使用:=
//全域變數必須使用var
var p1 = 83
var p2 int = 72
x := 42 + 7
```
